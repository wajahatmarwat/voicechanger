📋 Windows Setup Guide — Connect to ViciDial

▼
1

Install VB-Audio Virtual Cable

Download the free VB-Cable driver from vb-audio.com/Cable. Run the installer as Administrator and restart your computer. This creates a virtual audio device called CABLE Input / CABLE Output.

2

Open AccentFlow in Google Chrome

Open this page (index.html) in Google Chrome. Chrome is required for speech recognition to work. Allow microphone access when prompted.

3

Set Chrome Audio Output to VB-Cable

On Windows 10/11: Go to Settings → System → Sound → Volume Mixer (or App volume and device preferences). Find Google Chrome and set its Output device to CABLE Input (VB-Audio Virtual Cable). This routes AccentFlow's voice output through the virtual cable.

4

Open ViciDial in Microsoft Edge

Open ViciDial in Microsoft Edge (or another browser). In ViciDial's audio settings, set the microphone/input to CABLE Output (VB-Audio Virtual Cable). This makes ViciDial think the American accent TTS is your microphone!

5

Test Everything

Click the mic button above and start speaking. You should see your words transcribed, then hear them replayed in an American accent. On ViciDial, the customer should hear the American accent voice instead of yours. Adjust voice, speed, and pitch using the settings above.

6

Troubleshooting

No voices loading? Make sure you're using Chrome. Try refreshing the page.
Mic not working? Check Chrome's address bar for a microphone icon — click it to allow access.
ViciDial can't hear output? Verify VB-Cable is installed and Chrome's output is set to CABLE Input in Windows Sound Mixer.
Too much delay? Try increasing the speed slider to 1.2x or 1.3x to compensate.
